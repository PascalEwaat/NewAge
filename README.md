# NewAge

A lab for experimental web builds. Each experiment lives in its own folder
with its own README, dependencies, and dev setup.

**Live:** https://pascalewaat.github.io/NewAge/

## Experiments

| Experiment | Live | Description |
|---|---|---|
| [image-spring](./image-spring) | [demo](https://pascalewaat.github.io/NewAge/image-spring/) | Images rise from a spring in an inverted funnel and burst into particles on click. WebGL, Three.js, GSAP. |
